# Cube

This example demonstrates how to render a colored cube.

## Features

- Renders a 3D cube with distinct colors on each face
- Uses vertex buffer with position and color data for each vertex
- Implements a complete 3D rendering pipeline with depth testing

## Running

To build and run this example:

```shell
build.bat|sh 04_cube run
```
